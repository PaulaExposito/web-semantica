# Web semántica: SPARQL y Wikidata

## Usage

#### Cliente
```
cd client
npm install
npm run dev
```

Running on ```http://localhost:1234```

#### Servidor
```
cd server
npm install
npm run dev
```

Running on ```http://localhost:3000```


## Third-Party Technologies

* Parcel
* Vue
    * Naive UI
* NodeJS
* SparQL
* Wikidata
